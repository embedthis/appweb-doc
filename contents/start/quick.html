Appweb Quick Start

  1. Download

    First download Appweb on your system. You can download a binary package from https://embedthis.com/appweb/download.html.

  2. Installing Appweb

    Install Appweb on your system by following the instructions relevant for your system.

  3. Read the doc

    There is a wealth of documentation to read. Try the User's Guide and Developer's Guide first.
